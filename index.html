restack by adlawson

PHP 5.3+ datastructure library

Restack is powerful datastructure library based around the PHP5 SPL Datastructures. The main goal of this project is to offer a library with as little coupling and as much flexibility as possible while still providing key benefits of the SPL classes. Other goals include:
  • Easy to use directly and extend from
  • Item dependency tracking
  • Lightweight

Authors

Andrew Lawson http://github.com/adlawson
Peter Johnson http://github.com/missinglinknz

Download

You can download this project in either zip or tar formats, or clone the project with Git by running: 

$ git clone git://github.com/adlawson/restack

You could also add the library to your existing Git project by running: 

$ git submodule add git://github.com/adlawson/restack path/to/vendors/restack
$ git submodule update --init
